# WP - Gustavo Coimbra

## Como rodar

1. Tenha Docker e Docker Compose instalados.
2. Entre na pasta do projeto com o `docker-compose.yml`.
3. Rode:
   docker-compose up -d
4. Acesse no navegador:
   http://localhost:8080

## Parar o projeto

docker-compose down

---

Pronto, é só isso para rodar o WordPress no Docker.
